## Cabeçalhos (Linha 1)

| Coluna | Cabeçalho | Campo no Sistema |
|--------|-----------|------------------|
| A | Nº | attendanceNumber |
| B | Data/Horário | attendanceDate |
| C | Aluno | studentName |
| D | Curso | course |
| E | Semestre | semester |
| F | Ações | (não usado) |
| G | ASPECTOS OBSERVADOS | observedAspects |
| H | DIRETIVAS TOMADAS | directivesTaken |

## Formato dos Dados

- **Nº**: Número inteiro sequencial
- **Data/Horário**: Formato "DD/MM/YYYY HH:MM"
- **Aluno**: Texto (nome completo)
- **Curso**: Texto (nome do curso em maiúsculas)
- **Semestre**: Texto (ex: "1º", "2º", etc)
- **Ações**: Coluna vazia (placeholder)
- **ASPECTOS OBSERVADOS**: Texto longo
- **DIRETIVAS TOMADAS**: Texto longo

## Observações

- A planilha tem cabeçalhos coloridos (azul claro para colunas A-F, rosa para G, vermelho para H)
- Os dados começam na linha 2
- A coluna F (Ações) está presente mas não será preenchida automaticamente
